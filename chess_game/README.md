## 🎯 Цель работы

Изучить событийно-ориентированное программирование с использованием библиотеки Pygame и разработать игровое графическое приложение - классические шахматы.

---

## 📋 Функциональные требования

Приложение реализует следующий функционал:

1. **Игровая механика:**
   - Полная шахматная логика (все фигуры ходят по правилам)
   - Игра на двоих на одном компьютере
   - Переворот доски после каждого хода
   - Рокировка (короткая и длинная)
   - Превращение пешки при достижении последней линии
   - Проверка шаха и мата

2. **Меню и интерфейс:**
   - Главное меню: "Начать игру", "Таблица рекордов", "Справка", "Выход"
   - Таблица рекордов (сохраняет имя и количество ходов)
   - Справка с правилами игры
   - Подсветка выбранной фигуры и возможных ходов

3. **Звуковое сопровождение:**
   - Фоновая музыка
   - Звук хода фигурой
   - Звук взятия фигуры

4. **Анимации:**
   - Подсветка выбранной фигуры
   - Подсветка возможных ходов
   - Диалог выбора фигуры при превращении пешки

---

## 🏗 Архитектура приложения

### Model (board.py, pieces.py)
- Логика шахматной доски
- Классы всех фигур с правилами ходов
- Проверка шаха и мата
- Управление очередью ходов

### View (game.py, menu.py)
- Отрисовка доски и фигур
- Отображение меню и таблицы рекордов
- Визуальная подсветка
- Диалоговые окна

### Controller (game.py)
- Обработка событий (клики мыши, клавиши)
- Управление игровым циклом
- Связь между моделью и отображением

### Config (config.py)
- Загрузка настроек из JSON
- Хранение конфигураций игры

## 📊 Структура данных

### Класс Piece (базовый для всех фигур)

| Поле | Тип | Описание |
|------|-----|----------|
| color | str | Цвет фигуры (white/black) |
| row | int | Номер строки (0-7) |
| col | int | Номер столбца (0-7) |
| symbol | str | Символ фигуры |
| has_moved | bool | Флаг для рокировки |

### Классы фигур и их ходы:

| Фигура | Обозначение | Ходы |
|--------|-------------|------|
| Пешка | P/p | Вперёд на 1, бьёт по диагонали |
| Ладья | R/r | По горизонтали и вертикали |
| Конь | N/n | Буквой "Г" |
| Слон | B/b | По диагонали |
| Ферзь | Q/q | По всем направлениям |
| Король | K/k | На 1 клетку в любую сторону |
