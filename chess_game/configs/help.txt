ПРАВИЛА ИГРЫ В ШАХМАТЫ

Цель игры: поставить мат королю противника.

Основные правила:
1. Играют два игрока (белые и чёрные)
2. Белые ходят первыми
3. После каждого хода доска переворачивается для следующего игрока

Фигуры и их ходы:
- ♙ Пешка: ходит вперёд на 1 клетку, бьёт по диагонали
- ♖ Ладья: ходит по горизонтали и вертикали
- ♘ Конь: ходит буквой "Г"
- ♗ Слон: ходит по диагонали
- ♕ Ферзь: ходит по горизонтали, вертикали и диагонали
- ♔ Король: ходит на 1 клетку в любом направлении

Особые правила:
- Рокировка: король и ладья меняются местами при определённых условиях
- Превращение пешки: при достижении последней линии превращается в любую фигуру
- Взятие на проходе: специальное взятие пешки

Управление:
- Кликните по фигуре, затем по клетке для хода
- Нажмите ESC для возврата в меню

Превращение пешки:
Когда пешка достигает последней линии (1-й ряд для белых,
8-й ряд для чёрных), она может превратиться в любую другую
фигуру (кроме короля). При превращении появится диалог выбора.

Горячие клавиши в диалоге:
Q - Ферзь
R - Ладья
B - Слон
N - Конь
