## 📋 Функциональные требования

Приложение реализует следующий функционал:

1. **Управление записями:**
   - Добавление новых записей через диалоговое окно
   - Поиск записей по различным критериям
   - Удаление записей по условиям
   - Отображение всех записей в главном окне

2. **Постраничный вывод:**
   - Навигация: первая, предыдущая, следующая, последняя страницы
   - Настройка количества записей на странице (5-50)
   - Отображение информации о текущей странице и общем количестве записей

3. **Работа с файлами:**
   - Сохранение данных в XML-файл (DOM-парсер)
   - Загрузка данных из XML-файла (SAX-парсер)
   - Стандартные диалоги выбора файла

4. **Интерфейс:**
   - Главное окно с таблицей данных
   - Панель инструментов с кнопками быстрого доступа
   - Меню приложения
   - Дочерние диалоговые окна для добавления, поиска и удаления

---

## 🏗 Архитектура приложения (MVC)

### Model (model.py)
- Хранение данных о футболистах
- Бизнес-логика (добавление, поиск, удаление)
- Работа с XML (DOM для записи, SAX для чтения)

### View (view.py)
- Главное окно приложения
- Таблица для отображения данных
- Панель инструментов и меню
- Постраничная навигация
- Вызов диалоговых окон

### Controller (controller.py)
- Обработка действий пользователя
- Связь между Model и View
- Управление диалоговыми окнами

### Dialogs (dialogs.py)
- Диалог добавления записи
- Диалог поиска с отображением результатов
- Диалог удаления по условию

## 📊 Структура данных

### Класс FootballPlayer

| Поле | Тип | Описание |
|------|-----|----------|
| full_name | str | ФИО игрока |
| birth_date | datetime | Дата рождения |
| team | str | Футбольная команда |
| home_city | str | Домашний город |
| lineup | str | Состав (Основной/Запасной) |
| position | str | Позиция на поле |

---

## 🔍 Условия поиска и удаления

Поиск и удаление могут выполняться по любому из полей:
- По ФИО (поиск по части названия)
- По дате рождения (точное совпадение)
- По позиции (точное совпадение)
- По составу (Основной/Запасной)
- По футбольной команде (точное совпадение)
- По домашнему городу (точное совпадение)

**Важно:** Условия можно комбинировать. Например: найти всех нападающих из Зенита.
